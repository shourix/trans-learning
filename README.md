# トランス知識クイズ

変圧器（トランス）の理論・設計知識を確認するための学習用クイズアプリです。  
Streamlit 製。API 不要・ローカルで完結します。

## 機能概要

| 機能 | 説明 |
|------|------|
| フィルタ出題 | カテゴリ・問題形式・難易度を組み合わせて絞り込み |
| ランダム抽出 | フィルタ後の問題集からシャッフルして出題 |
| 即時フィードバック | 回答後にすぐ正解と解説を表示 |
| 結果サマリー | 正解数・正答率・評価（優秀 / 合格 / 要復習）を表示 |
| 誤答レポート | 間違えた問題を Markdown ファイルとしてダウンロード |

**問題カテゴリ（計 100 問）**

- 基本原理
- 損失・効率
- 温度上昇・熱設計
- 磁気設計
- 巻線設計・インピーダンス
- 電圧変動・変動率
- 試験・測定
- 総合計算

## セットアップ

### 動作確認済み環境

- Python 3.10 以上
- Windows 10 / 11

### 手順

**1. リポジトリをクローン（または ZIP 展開）**

```bash
git clone <リポジトリURL>
cd トランス学習
```

**2. 仮想環境を作成・有効化（推奨）**

```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Mac / Linux
source .venv/bin/activate
```

**3. 依存パッケージをインストール**

```bash
pip install -r requirements.txt
```

インストールされるのは `streamlit` のみです。

## 起動方法

```bash
streamlit run app.py
```

起動後、ブラウザが自動的に開きます（`http://localhost:8501`）。  
自動で開かない場合はアドレスを手動でコピーしてください。

**停止方法**：ターミナルで `Ctrl + C` を押します。

## 使い方

### 1. フィルタを設定する

画面左のサイドバーで出題条件を絞り込みます。

| 設定項目 | 説明 |
|----------|------|
| **カテゴリ** | 「基本原理」「磁気設計」など特定分野に絞れます |
| **問題形式** | 4択 / ○× / 穴埋め から選択。「すべて」で混在出題 |
| **難易度** | 入門 / 標準 / 応用。「すべて」で全レベル混在 |
| **問題数** | スライダーで 1〜30 問の範囲で指定。フィルタ後の問題数が上限になります |

> サイドバー下部に「対象問題数：XX 問」と表示されます。  
> フィルタを変えるたびにリアルタイムで更新されます。

### 2. 出題開始

`🎲 問題をランダムに選ぶ` ボタンを押すと、フィルタ条件に合う問題からランダムに抽出して出題が始まります。

### 3. 問題に答える

- ラジオボタンで選択肢を選び、`✅ 回答する` を押します
- 回答すると即座に正誤が表示され、解説が展開されます
- `次の問題 →` で次へ進みます

### 4. 結果を確認する

最終問題の後に `📊 結果を見る` を押すと結果画面が表示されます。

| 表示項目 | 説明 |
|----------|------|
| **正解数** | 「X / Y」形式 |
| **正答率** | パーセント表示 |
| **評価** | 80% 以上 → 優秀、60% 以上 → 合格、60% 未満 → 要復習 |

結果画面下部の「振り返り」セクションでは、全問の正誤・解説を一覧で確認できます。

### 5. やり直し

| ボタン | 動作 |
|--------|------|
| `🔄 もう一度同じ問題` | 同じ問題セットを最初から解き直す |
| `🎲 新しい問題を選ぶ` | サイドバーに戻り、再度ランダム抽出する |

## 誤答レポート

結果画面に誤答がある場合、`📥 誤答レポートをダウンロード（N 問）` ボタンが表示されます。

ボタンを押すと Markdown ファイルが保存されます。

**ファイル名の形式**

```
trans_quiz_report_YYYYMMDD_HHMM.md
```

**レポートの内容**

```
# トランス知識クイズ 誤答レポート

実施日時：2026-04-14 15:30
結果：7 / 10 正解（誤答 3 問）
```

## 問題の追加方法

`questions.json` に問題を追記するだけで、アプリに自動反映されます。  
アプリの再起動は不要です（ページをリロードするか、サイドバーの「対象問題数」が変わっていることを確認してください）。

> **注意**：`@st.cache_data` によりキャッシュされています。追加後に反映されない場合は、ブラウザ右上メニューの **「Clear cache」** を押すか、アプリを再起動してください。

### 問題データのフォーマット

```json
{
  "id": 101,
  "category": "カテゴリ名",
  "type": "multiple_choice",
  "difficulty": "basic",
  "question": "問題文",
  "choices": ["A. 選択肢1", "B. 選択肢2", "C. 選択肢3", "D. 選択肢4"],
  "correct": "A",
  "explanation": "解説文"
}
```

| フィールド | 値の例 | 説明 |
|-----------|--------|------|
| `id` | 101 | 一意の整数。既存の最大 id + 1 を使用 |
| `category` | `"磁気設計"` | 自由文字列。新カテゴリを追加するとサイドバーに自動追加 |
| `type` | `"multiple_choice"` | `multiple_choice` / `true_false` / `fill_blank` のいずれか |
| `difficulty` | `"basic"` | `basic`（入門）/ `standard`（標準）/ `advanced`（応用）のいずれか |
| `question` | `"問題文"` | 問題文 |
| `choices` | `["A. …", "B. …"]` | 4択は A〜D の 4 個、○× は `["○", "×"]` の 2 個 |
| `correct` | `"A"` | 4択は `"A"` / `"B"` / `"C"` / `"D"`、○× は `"○"` / `"×"` |
| `explanation` | `"解説文"` | 回答後に表示される解説 |

### ○× 問題のフォーマット例

```json
{
  "id": 102,
  "category": "基本原理",
  "type": "true_false",
  "difficulty": "basic",
  "question": "変圧器は直流電圧を変換することができる。",
  "choices": ["○", "×"],
  "correct": "×",
  "explanation": "変圧器は電磁誘導を利用するため交流専用。直流では磁束変化が生じないため誘導起電力が発生しない。"
}
```
